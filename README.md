# Campus-Predictor
# Project Overview
The Campus Placement Predictor is a Machine Learning web application that predicts whether a student is likely to get placed based on academic and skill-related parameters.
This project uses classification algorithms and is deployed using Streamlit for an interactive user interface.
# Problem Statement
Many students are unsure about their placement chances.
This project helps predict placement status based on:
CGPA
Number of Internships
Number of Projects
Communication Skills
The model predicts:
--> Placed
-->Not Placed
Along with probability percentage.
